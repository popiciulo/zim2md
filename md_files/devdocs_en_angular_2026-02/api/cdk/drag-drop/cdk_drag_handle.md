# CDK\_DRAG\_HANDLE

Injection token that can be used to reference instances of [`CdkDragHandle`](cdkdraghandle). It serves as alternative token to the actual [`CdkDragHandle`](cdkdraghandle) class which could cause unnecessary retention of the class and its directive metadata.

## API

```
const CDK_DRAG_HANDLE: any;
```
