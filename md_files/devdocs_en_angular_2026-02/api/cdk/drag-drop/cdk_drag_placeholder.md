# CDK\_DRAG\_PLACEHOLDER

Injection token that can be used to reference instances of [`CdkDragPlaceholder`](cdkdragplaceholder). It serves as alternative token to the actual [`CdkDragPlaceholder`](cdkdragplaceholder) class which could cause unnecessary retention of the class and its directive metadata.

## API

```
const CDK_DRAG_PLACEHOLDER: any;
```
