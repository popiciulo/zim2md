# CDK\_DRAG\_PREVIEW

Injection token that can be used to reference instances of [`CdkDragPreview`](cdkdragpreview). It serves as alternative token to the actual [`CdkDragPreview`](cdkdragpreview) class which could cause unnecessary retention of the class and its directive metadata.

## API

```
const CDK_DRAG_PREVIEW: any;
```
