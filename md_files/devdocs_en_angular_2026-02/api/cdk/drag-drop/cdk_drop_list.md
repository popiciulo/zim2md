# CDK\_DROP\_LIST

Injection token that can be used to reference instances of [`CdkDropList`](cdkdroplist). It serves as alternative token to the actual [`CdkDropList`](cdkdroplist) class which could cause unnecessary retention of the class and its directive metadata.

## API

```
const CDK_DROP_LIST: any;
```
