# CDK\_DROP\_LIST\_GROUP

Injection token that can be used to reference instances of [`CdkDropListGroup`](cdkdroplistgroup). It serves as alternative token to the actual [`CdkDropListGroup`](cdkdroplistgroup) class which could cause unnecessary retention of the class and its directive metadata.
