# CdkDragHandle

Handle that can be used to drag a CdkDrag instance.

## API

```
class CdkDragHandle implements AfterViewInit ,OnDestroy {
  element: any;
   get disabled(): boolean;
}
```

`any`

### disabled

`boolean`
