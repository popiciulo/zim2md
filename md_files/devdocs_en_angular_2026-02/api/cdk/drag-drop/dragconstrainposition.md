# DragConstrainPosition

Function that can be used to constrain the position of a dragged element.

## API

```
type DragConstrainPosition = (
  userPointerPosition: Point,
  dragRef: DragRef,
  dimensions: DOMRect,
  pickupPositionInElement: Point,
) => Point
```
