# CommonModule

Exports all the basic Angular directives and pipes, such as [`NgIf`](ngif), [`NgForOf`](ngforof), [`DecimalPipe`](decimalpipe), and so on. Re-exported by `BrowserModule`, which is included automatically in the root `AppModule` when you create a new app with the CLI `new` command.

## API

```
class CommonModule {
}
```
