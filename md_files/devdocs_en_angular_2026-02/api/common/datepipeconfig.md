# DatePipeConfig

An interface that describes the date pipe configuration, which can be provided using the [`DATE_PIPE_DEFAULT_OPTIONS`](date_pipe_default_options) token.

[DATE\_PIPE\_DEFAULT\_OPTIONS](date_pipe_default_options)

## API

```
interface DatePipeConfig {
  dateFormat?: string | undefined;
  timezone?: string | undefined;
}
```

### dateFormat

`string | undefined`

### timezone

`string | undefined`
