# HTTP\_INTERCEPTORS

A multi-provider token that represents the array of registered [`HttpInterceptor`](httpinterceptor) objects.

## API

```
const HTTP_INTERCEPTORS: InjectionToken<readonly HttpInterceptor[]>;
```
