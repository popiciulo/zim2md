# HttpClientJsonpModule

Configures the dependency injector for [`HttpClient`](httpclient) with supporting services for JSONP. Without this module, Jsonp requests reach the backend with method JSONP, where they are rejected.

Deprecation warning

[`withJsonpSupport()`](withjsonpsupport) as providers instead

## API

```
class HttpClientJsonpModule {
}
```
