# HttpClientTestingModule

Configures `HttpClientTestingBackend` as the `HttpBackend` used by `HttpClient`.

Deprecation warning

Add [`provideHttpClientTesting()`](providehttpclienttesting) to your providers instead.

## API

```
class HttpClientTestingModule {
}
```

## Description

Configures `HttpClientTestingBackend` as the `HttpBackend` used by `HttpClient`.

Inject [`HttpTestingController`](httptestingcontroller) to expect and flush requests in your tests.
