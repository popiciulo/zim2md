# ApplicationModule

Re-exported by `BrowserModule`, which is included automatically in the root `AppModule` when you create a new app with the CLI `new` command. Eagerly injects [`ApplicationRef`](applicationref) to instantiate it.

## API

```
class ApplicationModule {
  constructor(appRef: ApplicationRef): ApplicationModule;
}
```
