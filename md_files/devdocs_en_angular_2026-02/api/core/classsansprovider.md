# ClassSansProvider

Configures the [`Injector`](injector) to return a value by invoking a `useClass` function. Base for [`ClassProvider`](classprovider) decorator.

[Dependency Injection Guide](../../guide/di/dependency-injection)

## API

```
interface ClassSansProvider {
  useClass: Type<any>;
}
```

### useClass

`Type<any>`

Class to instantiate for the `token`.
