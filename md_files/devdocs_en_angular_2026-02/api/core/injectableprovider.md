# InjectableProvider

Injectable providers used in [`@Injectable`](injectable) decorator.

## API

```
type InjectableProvider = | ValueSansProvider
  | ExistingSansProvider
  | StaticClassSansProvider
  | ConstructorSansProvider
  | FactorySansProvider
  | ClassSansProvider
```
