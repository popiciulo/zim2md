# InputOptionsWithTransform

Signal input options with the transform option required.

## API

```
type InputOptionsWithTransform<T, TransformT> = Required<
  Pick<InputOptions<T, TransformT>, 'transform'>
> &
  InputOptions<T, TransformT>
```
