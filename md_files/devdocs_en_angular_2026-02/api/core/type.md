# Type

Represents a type that a Component or other object is instances of.

## API

```
const Type: FunctionConstructor;
```

## Description

Represents a type that a Component or other object is instances of.

An example of a [`Type`](type) is `MyCustomComponent` class, which in JavaScript is represented by the `MyCustomComponent` constructor function.
