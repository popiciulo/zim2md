# ValueSansProvider

Configures the [`Injector`](injector) to return a value for a token. Base for [`ValueProvider`](valueprovider) decorator.

## API

```
interface ValueSansProvider {
  useValue: any;
}
```

### useValue

`any`

The value to inject.
