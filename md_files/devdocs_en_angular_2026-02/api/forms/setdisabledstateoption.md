# SetDisabledStateOption

The type for CALL\_SET\_DISABLED\_STATE. If `always`, then ControlValueAccessor will always call `setDisabledState` when attached, which is the most correct behavior. Otherwise, it will only be called when disabled, which is the legacy behavior for compatibility.

[FormsModule#withconfig](formsmodule#withconfig)

## API

```
type SetDisabledStateOption = 'whenDisabledForLegacyCode' | 'always'
```
