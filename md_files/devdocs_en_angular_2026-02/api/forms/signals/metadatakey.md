# MetadataKey

Represents metadata that may be defined on a field when it is created using a [`metadata`](metadata) rule in the schema. A particular [`MetadataKey`](metadatakey) can only be defined on a particular field **once**.

## API

```
class MetadataKey<TValue> {
}
```
