# ANIMATION\_MODULE\_TYPE

A [DI token](../../core/injectiontoken) that indicates which animations module has been loaded.

## API

```
const ANIMATION_MODULE_TYPE: InjectionToken<"NoopAnimations" | "BrowserAnimations">;
```
