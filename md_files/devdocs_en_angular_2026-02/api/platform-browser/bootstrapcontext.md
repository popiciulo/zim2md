# BootstrapContext

A context object that can be passed to [`bootstrapApplication`](bootstrapapplication) to provide a pre-existing platform injector.

## API

```
interface BootstrapContext {
  platformRef: PlatformRef;
}
```

### platformRef

`PlatformRef`

A reference to a platform.
