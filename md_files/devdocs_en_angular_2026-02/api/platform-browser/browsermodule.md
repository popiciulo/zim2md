# BrowserModule

Exports required infrastructure for all Angular apps. Included by default in all Angular apps created with the CLI `new` command. Re-exports [`CommonModule`](../common/commonmodule) and [`ApplicationModule`](../core/applicationmodule), making their exports and providers available to all apps.

}
