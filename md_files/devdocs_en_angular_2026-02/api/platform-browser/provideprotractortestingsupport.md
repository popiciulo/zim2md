# provideProtractorTestingSupport

Returns a set of providers required to setup [Testability](../core/testability) for an application bootstrapped using the [`bootstrapApplication`](bootstrapapplication) function. The set of providers is needed to support testing an application with Protractor (which relies on the Testability APIs to be present).
