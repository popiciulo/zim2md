# InitialNavigationFeature

A type alias for providers returned by [`withEnabledBlockingInitialNavigation`](withenabledblockinginitialnavigation) or [`withDisabledInitialNavigation`](withdisabledinitialnavigation) functions for use with [`provideRouter`](providerouter).

## API

```
type InitialNavigationFeature = | EnabledBlockingInitialNavigationFeature
  | DisabledInitialNavigationFeature
```
