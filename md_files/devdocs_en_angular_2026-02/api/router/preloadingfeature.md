# PreloadingFeature

A type alias that represents a feature which enables preloading in Router. The type is used to describe the return value of the [`withPreloading`](withpreloading) function.

[withPreloading](withpreloading)[provideRouter](providerouter)

## API

```
type PreloadingFeature = RouterFeature<RouterFeatureKind.PreloadingFeature>
```
