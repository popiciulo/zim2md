# analytics disable

```
ng analytics disable[options]
```

```
ng analytics off[options]
```

Disables analytics gathering and reporting for the user.

## Options

### global

Configure analytics gathering and reporting globally in the caller's home directory.

#### Value Type

`boolean`

#### Default

`false`
